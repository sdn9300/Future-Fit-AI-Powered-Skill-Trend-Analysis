# Power BI Build Kit

This folder contains the exact copy-paste materials you need to finish the Power BI deliverable for Phase 6B.

## Files
- [powerbi_build_guide.md](powerbi_build_guide.md)
- [powerbi_measures.dax](powerbi_measures.dax)
- [future_fit_powerbi_theme.json](future_fit_powerbi_theme.json)
- [skill_trend_powerbi.pbix](skill_trend_powerbi.pbix)
- [skill_trend_dashboard.pdf](skill_trend_dashboard.pdf)

## What to do
1. Open Power BI Desktop.
2. Import `data/clean/primary_skills_powerbi.csv`.
3. Apply the theme JSON.
4. Create the measures from the DAX file.
5. Build the visuals using the guide.
6. Save the `.pbix` file here.
7. Export the report as PDF here.

The last two files are placeholder targets until you export the real Power BI report and PDF.
