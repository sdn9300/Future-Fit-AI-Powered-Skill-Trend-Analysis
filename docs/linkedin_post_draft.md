# LinkedIn Post Draft

I analyzed 50,000 AI and Data Science job postings from 2020 to 2026 to identify which skills are truly trending, then built a polished analytics workflow to show the results.

## Highlights

- The ML category makes up **101,975** of **224,605** cleaned skill mentions, or **45.4%** of the dataset.
- GenAI mentions are extremely rare in the LinkedIn validation sample: only **4** GenAI rows across **180,106** validation skill rows, all in `mid senior` postings.
- AWS is the most demanded skill with **20,638** mentions, and **AWS + GCP** is the strongest co-occurring pair with **7,892** joint mentions.

## Built with

Python, Pandas, Plotly, SciPy, Groq API, Streamlit, Power BI compatible export.
